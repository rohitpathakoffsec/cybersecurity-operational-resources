# File Sharing Tools

Services and applications for **securely sharing files** during security research,
incident response, and collaboration.

## Tools

- Send – https://send.vis.ee/
- Wormhole – https://wormhole.app/
- WeTransfer – https://wetransfer.com/
- OnionShare – https://onionshare.org/ *(requires local installation)*
- BiteBlob – https://biteblob.com/
