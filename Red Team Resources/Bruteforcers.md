# Bruteforce Tools

Tools used for **password cracking, credential brute-forcing, and high-volume
authentication testing** during penetration testing and security assessments.

## Tools

- **Hashcat** – High-performance password cracking tool.
- **Hydra** – Parallelized login cracker supporting numerous network protocols.
- **John the Ripper** – Popular password cracker for offline hash cracking.
- **Ophcrack** – Windows password cracker based on rainbow tables.
- **Turbo Intruder** – Burp Suite extension for sending large volumes of HTTP requests for testing.
